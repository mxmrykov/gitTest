##Приоритеты операций на Java

|  Операция    |Приоритет|
-------------------------
|!,+,- (Un.Op.)| Highest |

|*, /, %       |  Second |

|    +, -      |  Third  |

| <, <=, >, >= |  Fourth |

|    ==, !=    |  Fifth  |

|     &&       |  Sixth  |

|     ||       | Seventh |

| =(assigment) |  Lowest |
